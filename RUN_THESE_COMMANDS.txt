IMPORTANT: You must "activate" the virtual environment first!
This tells your computer to use the project's libraries (like decouple) instead of the system defaults.

STEP 1: Activate the Virtual Environment
   .\backend\venv\Scripts\activate

   (You should see "(venv)" appear at the start of your command line)

STEP 2: Create Admin User (Now it will work)
   python backend/manage.py createsuperuser

STEP 3: Start Server
   python backend/manage.py runserver

-------------------------------------------------------
If Step 1 gives an error like "script is disabled", use this command instead:
   backend\venv\Scripts\python backend\manage.py createsuperuser
