IMPORTANT: Keep the Backend running in your FIRST terminal!
Do NOT close it.

STEP 1: Open a NEW Terminal
   - In VS Code: Terminal -> New Terminal
   - Or open a new PowerShell window

STEP 2: Navigate to Frontend Folder
   cd frontend

STEP 3: Install Dependencies (Do this ONCE)
   npm install

STEP 4: Start Frontend Server
   npm start

This will open your browser at http://localhost:3000.
The map should load, and you should see the provinces!
